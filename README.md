Url Validator
=============

Description
=============
Modifikasi regex dari berbagai sumber untuk validasi beberapa format url.
Beberapa format yang bisa dikenali antara lain:
* [http://127.0.0.1:8080]
* [127.0.0.1]
* [google.com]
* [https://twitter.com]
* [bajigur.co.id:3128]

Usage Example
=============
python validate.py *input_url*
